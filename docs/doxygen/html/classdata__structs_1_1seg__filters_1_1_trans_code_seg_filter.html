Parameters
self
trans_code_index(int) the index (zero-based) of the transcriber code this filter will operate upon.
type_str(string) a string to use for the output of _get_filter_type_str()
desc_noun(string) used in get_filter_desc_str() to describe the object that this filter is filtering.
code_str_list(list) list of codes (that are valid for the transcriber code specified in trans_code_index) to allow through the filter (OR logic is used for multiple codes). (boolean=False) If True, the filter will invert its behaviour. If False, the filter acts normally.
db_id(int=None) this is the primary key id from the database table seg_filters. A value of None indicates that this filter is not yet in the DB.
