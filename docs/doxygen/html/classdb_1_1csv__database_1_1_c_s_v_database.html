Detailed Description

This class provides a way to convert a csv file into an in-memory database.

This allows you to do SQL-operations on the data, which in some circumstances can be much more convenient than implementing search (query), sort, or data modification methods in straight Python. The cost for doing this is a little extra time inserting all the data - however, this cost isn't usually too bad because in-memory databases are up to an order of magnitude faster than file-based ones. This class was implemented on a per-need basis, so some things (like delete(), or update for named columns added using add_column()) are not implemented yet. You can use the superclass's update() and delete() methods to do this yourself, or extend this class when the functionality is needed.

Sample usage:

1 #create a list of data types corresponding to the format of (a row in) your csv file
2 data_types = [str, int, float, bool, str, str, str]
3 
4 #this will create a single in-memory table, whose columns can be referred to by an integer index value (see select()
5 #call below)
6 db = CSVDatabase(data_types)
7 
8 #insert the data from the csv file - don't forget to skip over the header row
9 csv_reader.next()
10 for row in csv_reader:
11  db.csv_insert(row)
12 
13 #you can add extra named columns to the db that do not exist in the csv file
14 #Currently, these extra columns are not updatable - only selectable... This would not be too hard to change if
15 #future code requires it.
16 db.add_column('Extra column', bool)
17 
18 #update a value in column 3 (setting it to 2 wherever the value in column number 3 is equal to 5) - see
19 #Database.update() for a more detailed description. Right now the where_cond requires the
20 #column name, which is 'col<index>', making things less pretty than they could be...
21 #The first column has an index of 0.
22 db.update([3], where_cond='col3=?', params=[2, 5])
23 
24 #select some values from column with index 4 (where column index 3 is equal to the value 2)
25 rows = db.select([4], where_cond='col3=?', params=[2])
26 
27 #dump the selected data to another csv file
28 for line in rows:
29  csv_writer.writerow(line)

Adds a named column to the this database's single table.

Currently, these columns are not updatable using the csv_update method above. You'll need to do it manually using the superclass' update() method, or implement something new. The values in the newly added column are all set to NULL.

Parameters
self
