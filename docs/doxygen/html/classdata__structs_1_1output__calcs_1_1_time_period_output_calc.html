Searches through the internal dictionary for a key that intersects with the specified start and end times.

The dictionary is keyed by tuples of the form (start_time, end_time). This routine checks to see if the specified start and end range intersects with any key tuple already in the dictionary.

Parameters
self
start(float) an Utterance start time
end(float) an Utterance end time
Returns
(tuple) if an intersecting tuple is found, that tuple is returned. Else None is returned.
